# TicTacToe

Ein einfaches Tic-Tac-Toe-Spiel im Browser, erstellt mit HTML, CSS und JavaScript.  
Das Projekt bildet das klassische 3x3-Spielbrett digital nach und ermöglicht zwei Spielern, abwechselnd zu spielen.  
Das Spiel erkennt automatisch, wenn jemand gewonnen hat oder ein Unentschieden erreicht wurde.

## Über das Projekt

Das Projekt TicTacToe wurde entwickelt, um die Grundlagen der Webentwicklung mit HTML, CSS und JavaScript zu üben.  
Im Vordergrund steht die Arbeit mit dem DOM, die Umsetzung einer einfachen Spiellogik und die Gestaltung eines übersichtlichen Layouts mit CSS Grid.

---

## Features
- Zwei-Spieler-Modus (X und O)
- Gewinnererkennung und Anzeige von Unentschieden
- Neustart des Spiels per Button
- Responsives Design, auch auf mobilen Geräten spielbar

---

## Technologien
- **HTML5** für die Struktur  
- **CSS3** für das Layout und Design  
- **JavaScript (ES6)** für die Spiellogik und Interaktion  

---

## Verwendung

1. Öffne die Datei `index.html` im Browser.  
2. Spieler X beginnt und klickt auf ein Feld.  
3. Danach ist Spieler O an der Reihe.  
4. Sobald ein Spieler drei gleiche Symbole in einer Reihe, Spalte oder Diagonale hat, wird der Gewinner angezeigt.  
5. Mit dem Button „Neues Spiel“ kann das Spiel zurückgesetzt werden.
